# F14. 用量统计多维度查询与调用日志展示增强

## 8.14.1 需求说明

在用量统计页与调用日志页的数据查询中增加多维度筛选条件，并增强调用日志的展示字段。测试关注点在于筛选能力随用户视角收敛——普通用户仅在"我的用量"范围内筛选，且不可见"用户"筛选项；平台管理员的租户维度通过既有视角切换实现，不重复提供租户筛选器。各角色可见的筛选项集合不同，是权限测试的重点。

## 8.14.2 产品规则

- [PR-14.1] 所有筛选维度均服务于用量统计页与调用日志页的数据查询，由后端 API 支持并返回结果。测试关注：每个筛选维度单独提交后均能正确过滤返回对应数据，且筛选条件与列表、汇总指标区联动。
- [PR-14.2] 供应商来源仅作为模型展示列与聚合维度出现，不单独提供"供应商"筛选器。测试关注：筛选项集合中不出现"供应商"筛选项；但模型展示与 TOP 5 卡片中可见供应商名称。
- [PR-14.3] UA / IP 在调用明细页与调用日志查询页为 P0 必测；聚合统计视图的 UA / IP 维度为 P1，依赖后端统计能力，存在不支持的场景，需验证降级表现（明细仍可按 UA / IP 筛选，汇总指标区不纳入该维度）。
- [PR-14.4] 各筛选条件可组合使用，组合逻辑为 AND。测试关注：多筛选条件同时启用时，结果集为各条件的交集，非并集。

## 8.14.3 边界条件与异常场景

- **超月查询提示交互**：测试验证用户选择时间范围超过一个月时，前端在筛选条件区顶部展示提示条"平台只保留一个月记录，请缩小时间范围至最近一个月内查询"。提示条不阻断提交；前端自动将时间范围截断至最近一个月（保留用户选择的起止比例，如选了近 3 个月则截断到最后 1 个月）。用户忽略提示仍提交时，后端按实际时间范围返回可用数据，空结果不额外提示。测试关注：提示文案精确、截断后的起止时间、提交不阻断、空结果无额外弹窗。
- **TOP 5 钻展时间范围边界**：卡片钻展携带的时间范围为卡片统计口径的时间范围（当前页面筛选条件中的时间范围）。若用户在明细页手动调整时间范围超出卡片统计口径，筛选结果与卡片数据不再对应，属正常行为。测试关注：钻展后明细页时间筛选初值应等于卡片口径；在明细页修改时间后需提示或允许结果与卡片不一致，不报错。
- **汇总指标区空数据**：当前筛选条件无命中记录时，汇总指标区展示"暂无数据"占位，不展示 0 值。测试关注：空结果下各指标均显示"暂无数据"，而非 0；切换为有命中条件后恢复正常数值展示。
- **UA / IP 聚合 P1 降级**：当后端统计能力不支持 UA / IP 聚合时，调用日志明细页仍支持 UA / IP 筛选（P0 不降级），但汇总指标区不纳入 UA / IP 维度。测试关注：在降级场景下明细页 UA / IP 筛选功能正常；汇总指标区不出现 UA / IP 维度项，且不报错。
- **大量筛选条件组合性能**：用户组合超过 5 个筛选条件时，查询延迟可能增加；前端在延迟超过 3 秒时展示加载动画，不设硬性限制。测试关注：5 个以上筛选组合可正常返回结果（不阻断）；延迟超 3 秒出现加载动画。
- **跨月查询汇总指标**：汇总指标区统计口径覆盖所选时间范围内的所有月份，跨月不影响计算。测试关注：选择跨月时间范围后，汇总指标为各月数据之和/聚合值，不因跨月而缺失或重复。

## 8.14.5 页面交互

### 筛选条件区

- 各筛选项采用水平排列的下拉或日期区间选择器，支持组合筛选。测试关注：每个筛选项可独立选择与清除，且不与其他筛选项冲突。
- 时间范围超过一个月时，筛选区顶部展示提示条"平台只保留一个月记录，请缩小时间范围至最近一个月内"，同时前端自动将时间截断至最近一个月。测试关注：提示文案精确到字；截断后的起止时间符合"保留起止比例"规则。
- 筛选条件变更后，列表数据和汇总指标区自动联动刷新；搜索类筛选（用户、API Key、UA、IP）在输入停止 300ms 后触发查询。测试关注：防抖时延为 300ms（连续输入不应每次都发请求）；非搜索类筛选项变更应即时触发。

### 调用日志列表

- 缓存读写 Token 列、调用形式列为新增列，默认展示。测试关注：新列默认可见。
- 非流式调用的 TTFT 列展示为 `—`（em dash），不展示 0 或空值。测试关注：非流式记录 TTFT 列为 `—`，不为 "0"、空字符串或 null 文案。
- 列表上方汇总指标区随筛选条件联动实时刷新。测试关注：任意筛选条件变更后，汇总指标区与列表数据基于同一筛选口径同步刷新，无错位。

### 调用日志详情页

- 新增缓存读写 Token 展示区块。测试关注：含缓存读写记录的详情页正确展示对应数值；无缓存记录时按统一空值口径展示。
- 调用形式展示标签（流式 / 非流式），非流式 TTFT 展示为 `—`。测试关注：流式/非流式标签与记录的 `is_stream` 一致。
- 请求 / 响应原文：普通用户视角超过阈值（T16 定义）默认折叠，点击"展开完整内容"可查看全文；租户管理员 / 平台管理员视角默认展开。测试关注：普通用户长原文默认折叠且可展开；管理员视角默认展开；折叠阈值以 T16 确认值为准。

### TOP 5 分布卡片

- API Key TOP 5 卡片在"API 密钥"与"请求数"之间补"用户"列（API Key 创建人）。测试关注："用户"列位置在"API 密钥"与"请求数"之间，显示为 API Key 创建人。
- 模型 TOP 5 卡片模型列展示 `modelId + providerName`，同名不同供应商分两行。测试关注：同名不同供应商的模型分行展示且均带供应商名称；不合并为一行。
- 卡片行点击本期暂不跳转明细列表筛选（仅展示分布概览）。测试关注：点击卡片行无跳转、无报错，仅停留在分布概览视图。
- 二级钻展（卡片下钻到明细列表）时携带当前时间范围；明细页可独立调整时间筛选，不影响上层其他筛选条件。测试关注：钻展后明细页时间筛选初值等于卡片口径；在明细页修改时间筛选不影响用量统计页的其他筛选条件。

## 8.14.6 状态流转

本功能不涉及独立状态流转。筛选行为为即席查询，不改变数据状态。

## 8.14.7 验收标准

### 一、筛选功能（AC-14.1 ~ AC-14.3）

- [ ] **AC-14.1** Given 普通用户在用量统计页；When 查看筛选条件区；Then 不展示"用户"筛选项，其余筛选项可用。
- [ ] **AC-14.2** Given 用户在调用日志明细页；When 在 User-Agent 或 IP 筛选项输入值并查询；Then 列表按条件精确过滤。
- [ ] **AC-14.3** Given 两个不同供应商提供同名模型（如 `glm-5.1`）；When 用户在"实际命中模型"筛选项查看下拉；Then 两个同名模型分行展示，分别标注供应商名称。

### 二、调用日志展示（AC-14.4 ~ AC-14.6）

- [ ] **AC-14.4** Given 调用日志列表中存在含缓存读写 Token 的记录；When 用户查看列表和详情页；Then 缓存读取/缓存写入 Token 正确无误展示。
- [ ] **AC-14.5** Given 用户在调用日志列表页；When 变更任意筛选条件；Then 汇总指标区（总请求、成功、失败、限流、平均 RPM/TPM）随筛选条件联动刷新。
- [ ] **AC-14.6** Given 供应商返回 429 错误；When 统计汇总；Then 该请求计入限流；用量超额阻断计入失败、不计入限流。

### 三、TOP 5 卡片（AC-14.7 ~ AC-14.9）

- [ ] **AC-14.7** Given 用量统计页加载完成；When 查看 API Key TOP 5 卡片；Then 在"API 密钥"与"请求数"之间展示"用户"列，显示 API Key 创建人。
- [ ] **AC-14.8** Given 用量统计页加载完成；When 查看模型 TOP 5 卡片；Then 模型列展示 `modelId + providerName`，同名不同供应商分开呈现。
- [ ] **AC-14.9** Given 用户点击 TOP 5 卡片某一行；When 执行点击；Then 本期不跳转到明细列表筛选（仅展示分布概览）。

### 四、调用形式与原文（AC-14.10 ~ AC-14.12）

- [ ] **AC-14.10** Given 调用日志列表中存在流式和非流式调用；When 查看列表；Then 流式调用展示"流式"，非流式展示"非流式"；非流式调用 TTFT 列展示为 `—`。
- [ ] **AC-14.11** Given 用户选择时间范围超过一个月；When 提交查询；Then 前端展示"平台只保留一个月记录"提示条，并自动截断时间范围至最近一个月。
- [ ] **AC-14.12** Given 普通用户查看调用日志详情的请求/响应原文；When 原文长度超过阈值；Then 超出部分默认折叠，可点击"展开完整内容"查看；管理员视角默认展开。

### 五、钻展（AC-14.13）

- [ ] **AC-14.13** Given 卡片钻展到明细列表；When 执行钻展；Then 当前视图的时间范围随同应用为明细筛选项；明细页可独立调整时间筛选。

## 8.14.8 补充约束

- [约束-14.1] 性能 | 汇总指标区延迟超过 3 秒时前端展示加载动画；超时（10 秒）展示"数据加载超时，请缩小查询范围"提示，不返回部分结果 | 汇总指标区
- [约束-14.2] 功能完整性 | 调用日志列表新增列（缓存读写 Token、调用形式）的排序和导出需同步支持 | 调用日志列表
- [约束-14.3] 性能 | TOP 5 卡片本期不做前端缓存，每次切换时间范围或筛选条件均重新请求 | TOP 5 分布卡片
- [约束-14.4] 待确认 | 截断阈值 T16 待确认；默认值建议 10,000 字符，实际以产品确认值为准 | 调用日志详情页请求/响应原文
- [约束-14.5] 功能完整性 | 用户排行二级钻展加时间筛选，可按时间范围过滤用户用量明细 | 用户排行钻展
