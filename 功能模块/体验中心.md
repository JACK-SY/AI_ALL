## 模块定位与职责

体验中心是面向用户的**零代码模型在线体验入口**，用户无需编写代码即可在线调用大模型/小模型，验证模型效果，快速评估模型适用性。

核心职责：降低模型体验门槛，让用户"先用再接"。

## 子菜单结构

| 子菜单 | 说明 |
|--------|------|
| 大模型 | 大语言模型在线对话体验，支持多轮对话、深度思考、猜你想问 |
| 小模型 | 小模型（视觉/OCR 等）在线体验 |

## 页面功能概览（测试环境实测，2026-07-08）

- **模型选择**：支持按大模型/小模型切换，下拉选择具体模型
- **对话界面**：聊天式交互，支持文本输入、深度思考模式切换、多轮对话模式切换
- **历史记录**：右侧面板展示历史对话，支持搜索
- **猜你想问**：预设推荐问题，聚焦制造业/供应链场景
- **免责提示**：体验内容均由 AI 模型生成，不代表平台立场

## 与其他模块的关系

| 方向 | 关联模块 | 关系说明 |
|------|---------|---------|
| 上游 | 模型广场 | 体验中心可从模型广场"立即体验"入口跳转 |
| 上游 | 基础设置 | 底层调用依赖 API Key，但体验中心由平台内置 Key 承载，用户无需自备 |
| 下游 | 模型观测 | 体验调用会产生调用日志，计入用量统计 |
