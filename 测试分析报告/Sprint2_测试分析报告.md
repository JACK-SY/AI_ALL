## 3. 测试数据要求

| 字段 | 边界类型 | 边界值 | 测试点 |

### 3.2 等价类要求

| 输入类型 | 有效等价类 | 无效等价类 |
|---------|-----------|-----------|
| API Key | 8-200位字母数字组合 | <8位、特殊字符、中文、空值 |
| 费率 | 正数（最多4位小数） | 0、负数、超过4位小数、非数字 |
| 模型ID | 非空字符串、不重复 | 空值、重复ID、特殊字符 |
| 供应商类型 | 10个指定类型之一 | 非10个类型之一 |

### 3.3 测试环境要求

| 环境 | 用途 | 特殊要求 |
